# Beagle API tests


## Execute all tests

```
docker/dev/test
```


## Execute only one test suite

```
docker/dev/test "--testsuite [Unit/Integration/Feature]"
```


## Execute only one Test Class
```
docker/dev/test [path-to-test-class]
```


## Execute only one test
```
docker/dev/test "--filter [test-name]"
```
